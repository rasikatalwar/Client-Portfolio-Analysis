# Client Portfolio Analysis & Optimization Dashboard

This project simulates a real-world **Wealth Management** use case where a client portfolio is analyzed to evaluate **performance, risk, diversification**, and generate **data-driven rebalancing recommendations**.

The analysis is presented through an **interactive Streamlit dashboard** designed to be client-facing and easy to interpret for non-technical stakeholders.
