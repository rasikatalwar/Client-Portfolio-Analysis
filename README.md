# Client Portfolio Analysis & Optimization Dashboard

This project simulates a real-world **Wealth Management portfolio analysis** use case.
It focuses on evaluating client portfolio performance, risk, diversification, and
providing data-driven rebalancing recommendations.

## Key Features
- Portfolio performance and P&L analysis
- Risk metrics including volatility and Sharpe ratio
- Asset allocation and diversification analysis
- Rebalancing recommendations based on risk-return insights
- Interactive, client-facing dashboard built using Streamlit

## Tech Stack
- Python
- Pandas, NumPy
- Streamlit
- Plotly
